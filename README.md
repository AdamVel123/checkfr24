# FR24 Flight Finder

Веб-приложение на Python для поиска рейсов из FlightRadar24 по фильтрам:
- длительность рейса по расписанию (диапазон часов),
- страна/город/аэропорт вылета,
- страна/город/аэропорт прилёта,
- ICAO тип самолёта (например `B738`),
- авиакомпания,
- включение прошедших рейсов из кэша.

Также выводится **позывной (callsign)**.

## Установка

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\\Scripts\\activate
pip install -r requirements.txt
```

## Запуск

```bash
uvicorn app.main:app --reload
```

Откройте `http://127.0.0.1:8000`.

## Логика прошедших рейсов и кэша

- Все найденные рейсы сохраняются в SQLite (`flights_cache.db`).
- При включении галочки "Включать прошедшие рейсы" к результатам добавляются рейсы из кэша с признаком `is_past=true`.
- Перед каждым поиском кэш очищается от записей старше 5 дней.

## Важная валидация

Если не указан **ни один** фильтр, API вернёт ошибку с просьбой добавить хотя бы один фильтр.
